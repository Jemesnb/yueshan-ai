# 越山对话ai

**由mountain开发**

## 项目简介

“越山对话ai”是一个基于PHP实现的智能对话系统，旨在提供一个简洁、高效且**极度安全**的AI对话体验。该系统实现了全站密码访问保护，并支持全可视化的模型管理，确保您的API凭据万无一失。

## 核心安全特性

1.  **全站强制登录**：系统已集成统一身份验证模块。无论是对话主页（`index.php`）还是管理后台（`kjzz025247.php`），用户在访问前必须通过密码验证，有效防止非授权访问。
2.  **API 密钥脱敏保护**：在管理后台中，已保存的 API 密钥会以 `********` 形式显示。即使他人进入后台，也无法查看到真实的密钥明文。只有在您需要修改并输入新密钥时，系统才会进行更新。
3.  **全可视化配置**：所有操作均通过网页完成，无需手动编辑代码或 JSON 文件，降低了操作门槛及误操作风险。

## 主要功能

*   **多模型支持**：轻松集成并切换不同的 AI 对话模型（如 GPT 系列、Claude 系列等）。
*   **对话历史记录**：自动保存并显示当前会话的对话历史，方便用户回顾。
*   **响应式设计**：前端界面简洁现代，适配手机、平板及电脑等多种设备。
*   **模型管理后台**：提供直观的界面进行模型的增删改查操作。

## 技术栈

*   **后端**: PHP
*   **前端**: HTML, CSS, JavaScript
*   **API**: 支持任何兼容 OpenAI API 格式的对话模型接口

## Logo CSS 特色

“越山对话ai”的 Logo 融合了多种 CSS 特性，包括：
*   **呼吸灯阴影动画 (`logo-glow`)**：Logo 阴影在蓝色与红紫色间律动，赋予系统生命感。
*   **渐变边框效果**：利用 `linear-gradient` 打造多彩视觉边框。
*   **交互反馈**：鼠标悬停时，Logo 会平滑旋转 10 度并放大 1.1 倍。

## 安装与部署

1.  **下载项目**: 将项目文件下载到您的 Web 服务器根目录。
2.  **PHP环境**: 确保服务器支持 PHP 并启用了 `curl` 扩展。
3.  **访问系统**:
    *   **访问地址**: 浏览器打开 `index.php`。
    *   **默认访问密码**: `[REDACTED]`
    *   **安全建议**: 请务必在 `includes/auth.php` 文件中修改 `$admin_pass` 变量。
4.  **管理后台**: 
    *   访问 `kjzz025247.php` 进行模型配置。
    *   后台密钥显示为脱敏状态，确保隐私安全。

## 使用说明

*   **对话**: 登录后，在输入框输入问题并选择模型即可开始对话。
*   **注销**: 点击页面上的“退出登录”可安全退出当前会话。
*   **配置模型**: 在后台添加模型时，需填写显示名称、模型 ID、API 地址及密钥。
