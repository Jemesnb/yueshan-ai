# 越山对话ai

**由mountain开发**

## 项目简介

“越山对话ai”是一个基于PHP实现的智能对话系统，旨在提供一个简洁、高效的AI对话体验。该系统支持通过配置`models.json`文件来集成不同的AI模型API，实现灵活的模型切换和对话功能。前端界面设计简洁，响应式布局，适配不同设备。

## 主要功能

*   **多模型支持**：通过`models.json`配置，可轻松切换不同的AI对话模型（如GPT系列、Claude系列等）。
*   **对话历史记录**：自动保存并显示当前会话的对话历史，方便用户回顾。
*   **错误处理**：提供友好的错误提示，帮助用户了解API调用中出现的问题。
*   **简洁的用户界面**：采用现代设计风格，提供流畅的对话体验。
*   **模型管理后台**：`kjzz025247.php`提供了一个简单的后台界面，用于添加、编辑和删除AI模型配置。

## 技术栈

*   **后端**: PHP
*   **前端**: HTML, CSS, JavaScript
*   **API**: 支持任何兼容OpenAI API格式的对话模型接口

## 安装与部署

1.  **下载项目**: 将项目文件下载到您的Web服务器（如Apache, Nginx）的Web根目录。

2.  **配置 `models.json`**: 
    *   打开 `models.json` 文件。
    *   根据您使用的AI模型服务提供商，填写 `display_name` (显示名称), `model_id` (模型ID), `api_url` (API地址) 和 `api_key` (API密钥)。
    *   **重要提示**: `api_key` 是您的敏感信息，请务必妥善保管，不要在公共仓库中直接暴露。
    *   示例配置：
        ```json
        [
            {
                "display_name": "示例模型",
                "model_id": "gpt-3.5-turbo",
                "api_url": "https://api.openai.com/v1/chat/completions",
                "api_key": "YOUR_API_KEY_HERE" 
            }
        ]
        ```

3.  **PHP环境**: 确保您的服务器安装了PHP，并且启用了`curl`扩展，因为项目使用`curl`来调用外部API。

4.  **访问**: 在浏览器中访问 `index.php` 即可开始使用对话系统。访问 `kjzz025247.php` 可进入模型管理后台。

## 使用说明

*   **对话**: 在主页的输入框中输入您的问题，选择合适的AI模型，点击发送按钮即可开始对话。
*   **新对话**: 点击页面右上角的“新对话”按钮可以清空当前对话历史，开始新的对话。
*   **模型管理**: 访问 `kjzz025247.php`，您可以添加新的AI模型配置，编辑现有模型的信息，或者删除不再使用的模型。请确保填写的API信息准确无误。
